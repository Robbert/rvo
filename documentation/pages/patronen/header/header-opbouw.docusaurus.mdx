# Header opbouw

Een header bestaat uit:

- Links een menu item naar de home met de titel van desbetreffende regeling/website indien van toepassing
- Rechts altijd een uitloggen button als de gebruiker is ingelogd
- Rechts de naam van de klant (evt. met relatienummer patroon) die verwijst naar een profielpagina
