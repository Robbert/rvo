<p>
  Als de bezoeker bij een applicatie van RVO inlogt start de bezoeker een sessie. Deze sessie heeft voor de veiligheid
  vaak een beperkte duur. Het is belangrijk om de bezoeker hiervan op de hoogte te brengen. Dit betekent:
</p>

- Toon altijd de resterende sessietijd.
- Toon een popup met een keuze om de sessie te verlengen, een minuut voor de sessie verloopt.
- Toon een popup met de boodschap 'Uw sessie is verlopen'. Leg uit wat de bezoeker kan doen en toon een linkje naar de inlog-pagina.
- Idealiter komt de bezoeker na opnieuw inloggen weer terug op de eerdere pagina.
